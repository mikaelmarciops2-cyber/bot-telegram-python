# Bot Telegram em Python

Projeto simples de bot para Telegram feito em Python.

## Funcionalidades

- Comando /start
- Comando /ajuda
- Respostas automáticas
- Integração com API do Telegram

## Tecnologias usadas

- Python 3
- python-telegram-bot

## Como executar

Instale as dependências:

```bash
python -m pip install -r requirements.txt

Adicione seu token no arquivo:

TOKEN = "SEU_TOKEN_AQUI"

Execute:
python main.py

Estrutura do projeto:
bot-telegram-python/
│
├── main.py
